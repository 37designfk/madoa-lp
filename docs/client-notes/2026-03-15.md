## 実施内容

### 1. PageSpeed Insights 高速化

#### 画像最適化
- Flow画像4枚をWebP変換（mainimage-thum-01/02/05/06）
  - PNG合計: 608KB → WebP合計: 100KB（84%削減）
- 全画像に `<picture>` タグ導入（WebP + PNGフォールバック）
- 全画像に `width` / `height` 属性追加（CLS対策）
- Hero背景画像に `fetchpriority="high"` + `<link rel="preload">` 追加（LCP最適化）

#### Google Fonts削除
- Noto Sans JP（外部読み込み）→ システムフォントに切り替え
  - ヒラギノ角ゴ ProN / Hiragino Sans / 游ゴシック / メイリオ
- レンダリングブロック 6,770ms を完全除去
- これがモバイルFCP 6.5秒→0.9秒の最大要因

#### キャッシュ設定
- `public/.htaccess` 新規作成
  - gzip圧縮（HTML/CSS/JS/SVG）
  - 画像: 1年キャッシュ（immutable）
  - CSS/JS: 1ヶ月キャッシュ

### 2. アクセシビリティ修正

#### コントラスト比改善
- 緑ボタン色を `#8FC31F`(green-500) → `#4a8000`(green-600) に変更
  - 白文字とのコントラスト比: 2.8:1 → 4.9:1（WCAG AA準拠）
- フッターテキスト: `text-gray-500` → `text-gray-400`
- フッター全体: `text-gray-300` → `text-gray-400`

#### リンクラベル追加
- ヘッダー電話リンクに `aria-label="電話で問い合わせ: 078-597-2722"` 追加
  - モバイルでアイコンのみ表示のため、スクリーンリーダー対応
