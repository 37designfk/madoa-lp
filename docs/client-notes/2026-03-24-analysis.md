## 戦略的な提言

### 1. 内窓 × 補助金を最大化すべき
- 内窓はMQ貢献No.1かつM率50%超
- 補助金（まどりのべ）対象であり、顧客の導入ハードルが下がる
- **補助金専用LP + チラシで集客を強化** → 今回の打ち合わせで決定済み

### 2. madoa（自社集客）の比率を上げるべき
- 自社集客のM率47%は下請け/紹介より10pt高い
- 現在は全体の12.6%だが、**20%以上を目指す価値あり**
- LP + 広告 + 補助金訴求で自社集客を増やす戦略が正しい

### 3. 季節変動への対策
- 1〜2月の閑散期対策として、補助金案件（内窓）の受注を前倒しで獲得
- 補助金は年度切り替わりで需要が集中するため、3月に向けた先行営業が有効

### 4. 高M率工種の拡大
- 硝子割れ替え（M率66.4%）、網戸張替（67.4%）、浴室戸交換（65.6%）は高効率
- これらは「ついで提案」でmadoa経由の顧客にクロスセルできる

### 5. 大規模修繕のM率改善
- PQ 3,812万もあるのにM率20.7%は低い
- 外注費の見直しや、自社施工比率を上げることで改善の余地あり
